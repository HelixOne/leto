
# Leto

Leto is Framework, for building real-time, web, mobile, desktop, console, and iot appications. 	

## Stack
* Angualr2
* Koa
* RethinkDB/Horizon
* Electron (Windows, Mac, Linux)
* Cordova (iOS)
* Crosswalks (Android)
* Universal Windows Platform (Windows Store)
